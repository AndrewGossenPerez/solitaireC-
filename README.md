
A cheeky bit of Solitaire never hurt no one. 
This was made in less than a day, don't expect perfection. 
Requires SFML.
--

1) Installation:
 
   MacOs: brew install sfml
   
   Ubuntu: sudo pacman -S sfml
   
   Windows: vcpkg install sfml -> vcpkg integrate install
   
2 ) Build :

   make
   
3) Run the game :
   
   ./solitaire
   
   Or :
   
   makerun
